# Onboarding T.Group (HTML)

Um deck em HTML/JS, pronto para subir no GitHub e publicar no Vercel como site estático.

## Como usar

1. **Edite o conteúdo** em `content.js`.
2. **Troque logos e fotos** em `/assets`:
   - Logos das empresas: `/assets/logos/` (substitua os SVGs placeholders pelos arquivos reais).
   - Foto dos sócios: `/assets/people/` (substitua `placeholder-person.svg` por fotos reais; mantenha o mesmo nome no `content.js`).
3. Abra `index.html` no navegador para testar.
4. Suba para o GitHub e publique no Vercel.

## Atalhos

- ← → navegar
- espaço avançar
- O: abrir/fechar *overview*

## Dicas rápidas

- Se o deck for exibido em TV, coloque o navegador em tela cheia (F11).
- Para otimizar carregamento, prefira imagens WebP.
- Mantenha cada slide com no máximo 6 bullets.
